# MacOSTestWebApp
Setup and run a website with a database with some 100 lines of code.

# Prerequisites on MacOS
__None__

Nothing needs to be installed on MacOS. 
We rely on 
- Python 2.7
- SQLite3
- CGIHTTPServer 
provided out of the box on any MacOS platform.

- HTML
- Javascript
- Fetch API
